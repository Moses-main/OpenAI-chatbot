# Simple AI Chatbot

A lightweight, browser-based chatbot that uses the OpenAI API to generate responses. Built with pure HTML, CSS, and JavaScript.

## Features

- Clean, responsive interface
- Real-time chat with AI
- No backend required (runs entirely in the browser)
- Easy to customize and extend

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd chatbot
   ```

2. **Add your OpenAI API Key**
   - Open `index.html` in a text editor
   - Replace `'YOUR_OPENAI_API_KEY'` with your actual OpenAI API key
   - Save the file

3. **Run the Application**
   - Open `index.html` in a modern web browser
   - Start chatting!

## Usage

1. Type your message in the text area
2. Click "Send" or press Enter to send your message
3. The AI will respond in the chat window

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- OpenAI API key (get one at [OpenAI Platform](https://platform.openai.com/))

## Note

- For security reasons, it's recommended to use environment variables or a backend service to store your API key in production
- This is a frontend-only implementation for demonstration purposes
